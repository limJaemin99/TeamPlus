문의하기
일반적인 질문

저희 TeamPlus 에 로그인 또는 회원가입을 하신 이후에 상단에 프로젝트를 클릭 하시면 사이드바에서 프로젝트 생성이 있습니다. 프로젝트 생성 메뉴를 클릭하시면 쉽게 생성하실 수 있습니다.

팀 프로젝트를 생성하신 이후에 초대 코드와 프로젝트 비밀번호를 팀원분들께 전달해주시면 전달받은 팀원분들이 프로젝트 참가하실때 입력하고 들어오시면 초대가 가능합니다. 만약, 프로젝트 초대 코드를 잊으실 경우 프로젝트 개요에서 확인하실 수 있습니다. 또한, 팀원은 제한 수 없이 가능하며 초대코드와 프로젝트 비밀번호만 정확하게 입력하시면 팀원을 무제한으로 초대하실 수 있습니다.

NEW TODO에 있는 리스트는 고객님께서 등록하신 이후 1시간 뒤에 자동으로 진행전으로 업데이트가 되고 진행전 TODO에서 스크롤바를 내리시면 확인하실 수 있습니다.

네, 저희 TeamPlus는 전부 무료로 사용 가능하며 유료 결제 화면이 뜰 경우 바로 닫아주시기 바랍니다.
계정 관리

회원가입은 TeamPlus 메인 화면 오른쪽 상단에서 회원가입 메뉴 클릭 후 양식에 맞게 입력해주시면 회원가입이 완료됩니다.

로그인은 TeamPlus 메인 화면 오른쪽 상단에서 로그인 메뉴 클릭 후 양식에 맞게 입력해주시면 로그인이 완료됩니다.

비밀번호를 잊으셨을 경우 비밀번호 변경이 가능합니다.회원가입 하실 때 입력하셨던 이메일로 TeamPlus 인증번호로 인증을 하시면 비밀번호를 쉽게 변경하실 수 있습니다.

네, 회원 정보 수정도 가능합니다. 닉네임은 다른 사용자들과 겹치면 안되기 때문에 중복검사가 필수로 들어갑니다. 또한, 본인이 아닌 다른 사람이 악의적으로 정보를 수정하려는 경우를 대비하여 정보를 수정 하실 때 이메일 인증이 필수로 들어갑니다.
개인 정보 보호 및 보안

이메일 인증을 진행하는 이유는 개인 정보 보호를 위해 진행을 하고 있고 본인이 아닌 다른 사람이 악의적으로 계정을 해킹하는 경우의 수를 생각하고 진행하고 있습니다.

회원가입 할때 입력한 정보는 절대 노출되는 일 없게 철저히 관리중에 있습니다.

초대코드를 프로젝트 생성할 때마다 각각 다른 코드를 제공하고 있고 그럼에도 불구하고 초대하지 않은 팀원이 들어왔을 경우 팀원 내보내기 기능과 비밀번호 변경 기능이 있습니다. 비밀번호를 더욱 더 보안성있게 입력해서 팀원들께 공유해주시길 바랍니다.

이메일을 잊으셨을 때는 저희 TeamPlus에 이메일을 보내주시면 신속히 답변 드리겠습니다.
